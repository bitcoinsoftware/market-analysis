# market-analysis

### Installation:

First install the required libraries
on **Ubuntu 14.04** just type in the terminal:
> `$ sudo apt-get install python-beautifulsoup python-pip`
> `$ sudo pip install trans`

### Dependencies

| `Library` | `Version`|
|---| --- |
| `BeautifulSoup` | `3.2.1` |
| `mock` | `2.0.0` |
